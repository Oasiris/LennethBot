# Lenneth Bot

## Libraries

* Eris
* Node 10
* Babel

## Setup Checklist

 * [ ] Filled out `.env`
 * [ ] Ran `npm install`
 * [ ] Added `secret/service-account.json` for your Firebase project 
 * [ ] Set up Firebase

#### Setting up Firebase

```sh
npm install -g firebase-tools
firebase login
firebase list
firebase use <project-to-use>
```
